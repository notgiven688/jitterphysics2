Class PointOnLine

Namespace
Jitter2.Dynamics.Constraints
Assembly
Jitter2.dll

Constrains a fixed point in the reference frame of one body to a line that is fixed in the reference frame of another body. This constraint removes one degree of translational freedom; two if the limit is enforced.
