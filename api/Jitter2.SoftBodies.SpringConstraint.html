Constrains two bodies to maintain a target distance between anchor points, applying spring-like forces. Removes one translational degree of freedom along the line connecting the anchors.

Remarks

The spring behavior is controlled by Softness and Bias, which can be set directly or computed from physical parameters using SetSpringParameters(float, float, float).

Remarks

Computes local anchor offsets from the current body positions. Default values: Softness = 0.001, Bias = 0.2. The TargetDistance is set to the initial distance between the anchors.
